# Карго жеткізу веб-сайты — талаптар тізімі

**Жоба:** Байерлік қызмет (карго жеткізу)  
**Орта:** VS Code, PowerShell, `C:\Users\user\cargo`

---

## Функционалдық талаптар

| № | Талап | Статус |
|---|--------|--------|
| 1 | Пайдаланушы аутентификациясы (тіркелу, кіру, шығу) | ✅ |
| 2 | Рөлдер: қарапайым пайдаланушы және әкімші | ✅ |
| 3 | Таңдалған тақырыптың негізгі функционалдығы (карго тапсырыс, баға, маршрут) | ✅ |
| 4 | Контентті басқаруға арналған админ-панель | ✅ |
| 5 | Адаптивті интерфейс (десктоп + мобильді) | ✅ |
| 6 | Дерекқормен интеграция (сақтау) | ✅ |
| 7 | RESTful API (фронтенд ↔ бэкенд) | ✅ |
| 8 | Валидация және қателерді өңдеу (клиент + сервер) | ✅ |
| 9 | Негізгі деректерді іздеу және сүзгілеу | ✅ |
| 10 | Сұрыптау және пагинация | ✅ |
| 11 | Логирование және мониторинг | ✅ |
| 12 | Автоматтандырылған тестілеу (unit + integration) | ✅ |
| 13 | Қауіпсіздік (хэштеу, sanitization, SQL/XSS/CSRF қорғанысы) | ✅ |
